# Pakistan 2050 — A Vision for the Future

> A futuristic, premium static website for a developer portfolio project.

## 🚀 Live Preview
Deploy to GitHub Pages or Netlify — fully static, no backend required.

## 📁 File Structure
```
/project
├── index.html          ← Main HTML (all sections)
├── css/
│   └── style.css       ← All styles (CSS variables, glassmorphism, animations)
├── js/
│   └── script.js       ← Animations, counters, canvas, interactions
├── images/             ← Add your images here
└── README.md
```

## ✨ Features
- Animated star/nebula canvas background
- Typing effect hero text
- Scroll-reveal animations (fade up / left / right)
- Animated counters
- Progress bars
- Animated GDP bar chart
- Wind turbine, solar orbit, hydro drop animations
- Hyperloop pod animation in gallery
- Card tilt effect (desktop)
- Cursor glow (desktop)
- Mobile hamburger menu
- Back-to-top button
- Gallery click interaction
- Contact form with success feedback
